# Kabaddi Arena

Kabaddi Arena is a comprehensive Android application designed for Kabaddi enthusiasts, coaches, and players to manage matches, track player performance, and analyze game statistics.

## Features

- **Live Match Logger:** Record match events in real-time with an intuitive interface.
- **Performance Analytics:** Detailed summaries and performance cards for players.
- **Interactive Kabaddi Court:** Visual representation of the court for better tactical understanding.
- **Match History:** Keep a local record of all played matches using Room database.
- **Modern UI:** Built with Jetpack Compose for a smooth and responsive user experience.

## Tech Stack

- **Language:** Kotlin
- **UI Framework:** Jetpack Compose
- **Architecture:** MVVM (Model-View-ViewModel)
- **Dependency Injection:** Hilt
- **Database:** Room
- **Navigation:** Compose Navigation

## Getting Started

### Prerequisites

- Android Studio Ladybug (or newer)
- JDK 11
- Android SDK 29+

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/KabaddiArena.git
   ```
2. Open the project in Android Studio.
3. Sync the project with Gradle files.
4. Run the app on an emulator or a physical device.
